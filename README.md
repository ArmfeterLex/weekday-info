# Weekday Info

Модуль для консольных приложений на C#, определяющий название дня недели по номеру и его тип (рабочий/выходной).

<img width="570" height="152" alt="Снимок экрана 2026-03-13 203459" src="https://github.com/user-attachments/assets/a7f8e3f2-7e92-4c83-8eb4-395ba5dac1df" />

## Функциональность

- Ввод номера дня недели (1-7)
- Преобразование в название
- Определение типа дня (рабочий/выходной)
- Обработка некорректного ввода

## Пример работы

Введите номер дня недели: 1, 2, 3, 4, 5 ,6 ,7
3
Среда
Рабочий день

Введите номер дня недели: 1, 2, 3, 4, 5 ,6 ,7
7
Воскресенье
Выходной

Введите номер дня недели: 1, 2, 3, 4, 5 ,6 ,7
9
Некорректный номер дня недели

## Куда можно встроить

- В системы планирования
- В календарные приложения
- В табель учёта рабочего времени
- В расписания

## Требования

- .NET Core / .NET Framework
- Консольное приложение

## Примечание

Для рабочих дней программа ожидает нажатия клавиши, для выходных — нет.
